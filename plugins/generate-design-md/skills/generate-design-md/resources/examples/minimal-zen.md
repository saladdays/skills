# Design System: Karesansui（枯山水）

## TL;DR

- 北極星: "画面に触れた瞬間、呼吸が深くなる" — 砂紋のように静かで余白に満ちた瞑想タイマー
- カラー: 砂と墨のモノトーン、アクセントは雫 (#7A9E9F) を1画面1箇所のみ
- タイポ: Noto Sans JP、3段階、16px基準、Weight上限400
- スペーシング: 余白がデザインそのもの。タイマー周囲は128px以上確保
- 避けること: グラデーション、blur、バッジ/通知、font-weight 500以上

## Design Principles

### Creative North Star

**Metaphor:** "画面に触れた瞬間、呼吸が深くなる" — 枯山水の石庭。砂紋は静かに広がり、雫がひとつ落ちるだけで空間に意味が生まれる。

**This means:**
- 余白はデザイン要素ではなく、デザインそのもの（Ma / 間）
- 各セッションは一度きりの体験として扱い、統計やゲーミフィケーションで「比較」を生まない（Ichigo-ichie / 一期一会）
- 完全な対称を避け、わずかな非対称で生命感を出す。計算された不完全さが自然な美を生む（Fukinsei / 不均斉）
- 画面の60%以上を常に余白として確保する
- 操作は最小限。UIはほぼ存在せず、画面全体が「間」として機能する

**This does NOT mean:**
- 装飾的な和風テクスチャやイラストを加える
- アニメーションを多用して「動く禅」を演出する
- 静寂を理由にフィードバックを省略する（静寂とは「何も足さないこと」であり「何も返さないこと」ではない）

### When Principles Conflict

- **間 vs 情報表示**: 残り時間の表示が必要な場合でも、画面の60%以上を余白として確保した上で配置する
- **不均斉 vs 一貫性**: 非対称はレイアウトの「重心」に適用し、インタラクション要素の配置には適用しない

### Global Constraints

| Constraint | Alternative | Reasoning |
|------------|-------------|-----------|
| グラデーション背景を使わない | 単色 `suna-100` を全面に使う | 視覚的刺激を最小化するため |
| Glassmorphism / blur を使わない | 透明度なしのソリッドカラー | 瞑想中の焦点ボケは不安を誘発するため |
| バッジ・通知ドットを置かない | セッション後に静かなサマリーを1画面で表示 | 達成圧力を排除するため |
| `font-weight` 500以上を使わない | サイズかスペーシングで強調する | Bold は視覚的な「叫び」になり静寂を壊すため |

## Color Strategy

### Palette

| Token | Value | Role | Do | Don't |
|-------|-------|------|----|-------|
| `suna-100` | `#F5F0EB` | 背景・砂 | 全画面の背景に使う | 部分的なカードの背景には使わない |
| `suna-200` | `#E8E0D8` | 区切り・砂紋 | セクション境界の微細な区切り線に | コンテナの塗りに使わない |
| `sumi-700` | `#3A3632` | テキスト・墨 | 本文テキスト、タイマー数字に | 大面積の塗りに使わない |
| `sumi-400` | `#8C8580` | 補助テキスト | ラベル、キャプションに | 主要な情報には使わない |
| `shizuku` | `#7A9E9F` | アクセント・雫 | 開始ボタンのみに使用（1画面1箇所） | 2箇所以上に同時使用しない |

### Color Rules

- **One-Drop Rule**: アクセント色 `shizuku` は1画面に1要素のみ。雫が石庭に落ちるように、視線の起点を1つだけ作る
- **Ink Hierarchy**: テキストは `sumi-700`（主）→ `sumi-400`（従）の2段階のみ。3段階目が必要なら情報設計を見直す
  - 例外: タイマー数字は `sumi-700` を `opacity: 0.87` で使用し、背景に沈む印象を与える

## Typography

### Scale

| Token | Font | Size | Weight | Line Height | Use |
|-------|------|------|--------|-------------|-----|
| `timer-display` | Noto Sans JP | 72px | 200 (Thin) | 1.0 | タイマー数字 |
| `body` | Noto Sans JP | 16px | 300 (Light) | 1.8 | 説明文・ガイド |
| `caption` | Noto Sans JP | 13px | 300 (Light) | 1.6 | 補助情報・ラベル |

### Typography Rules

- **Breath Rhythm Rule**: `line-height` は1.6以上を維持する。行間は「呼吸の間」として機能するため
- **Weight Ceiling**: `font-weight` は最大400。強調が必要な場合はサイズかスペーシングで解決する（例外なし）

## Spacing & Layout

### Scale

| Token | Value | Use |
|-------|-------|-----|
| `space-1` | 4px | アイコンとラベルの間 |
| `space-2` | 8px | 関連要素のグルーピング |
| `space-4` | 16px | セクション内の要素間 |
| `space-8` | 32px | セクション間 |
| `space-16` | 64px | 画面上部の余白・タイマー周囲 |
| `space-32` | 128px | タイマー画面の上下マージン |

### Spacing Rules

- **Emptiness-First Rule**: レイアウトは「何を置くか」ではなく「どれだけ空けるか」から設計する。最小マージンは `space-8`（32px）
- **Breathing Zone**: タイマー数字の上下には必ず `space-32`（128px）以上を確保し、数字が「浮いている」印象を作る
- **Touch Target**: タップ領域は48x48dp以上を確保するが、視覚的なボタンサイズは最小限に抑える（ゴーストボタン推奨）
