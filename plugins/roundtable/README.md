# roundtable

多分野の世界最高レベルの専門家を動的に選定し、構造化された議論で多角的な評価・提言をまとめるスキルです。

## 特徴

- **動的な専門家選定**: 議題に応じて最適な専門家パネルを自動構成
- **構造化された議論**: 独立分析→対立点の議論→統合の3段階プロセス
- **少数意見の保護**: 見逃してはいけない指摘を構造的に拾い上げる
- **Devil's Advocate**: 必ず1名は批判者ロールを配置
- **あらゆるドメイン**: コード、デザイン、事業戦略、組織設計、何でも対応

## 使い方

```
/roundtable:convene このWebサイトを評価して: https://example.com
/roundtable:convene 新規事業の戦略をレビューして（ドキュメントを添付）
/roundtable:convene このコードのアーキテクチャを評価して
/roundtable:convene チームの組織構造について議論したい
```

## 議論の深度

| 深度 | 専門家数 | ラウンド | 用途 |
|---|---|---|---|
| Quick | 3名 | 1 | 個人プロジェクト、軽い相談 |
| Standard | 4名 | 最大2 | 一般的な評価・レビュー |
| Deep | 5名 | 最大2 + セカンドチャンス | 事業判断、高リスクな意思決定 |
